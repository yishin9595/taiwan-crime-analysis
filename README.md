# Crime data analysis in Taiwan (2020–2024)

## 專案介紹
- **研究目的**：觀察近五年(2020–2024)台灣犯罪率狀況，作為租屋、購屋族以及相關單位治安部署參考
- **資料來源**：警政署犯罪資料、人口統計資料、新北臺北實價登錄
- **研究工具**：BigQuery、Google Sheets、Excel、Google Slides
- **使用情境**：
  - BigQuery：進行 SQL 資料查詢
  - Google Sheets、Excel：製作圖表
  - Google Slides：製作分析簡報
- **主要發現**：
  - 近五年治安趨近穩定，但 2024 年反彈；住宅竊盜、強制性交需提高警戒，需針對熱點縣市加強防治、連假前超前部署
  - 基隆、桃園、嘉義市為近五年犯罪率前三高，近年再度升溫，犯罪以毒品與竊盜為主，需強化宣導與在地防治
  - 租屋購屋族需避開萬華、三重區，北投、文山、內湖與士林區性價比高，可以優先考慮

---

## 分析前架構大綱
在進行分析前有先規劃以下分析面向，確保洞察出的數據能轉為實際行動：

1. **年度與月份犯罪趨勢**
   - 分析方向：觀察治安風險是否惡化，以及是否存在季節性規律（如寒暑假、春節）
   - 衡量指標：每萬人案件率、月份總案件數
   - 行動建議：判斷高風險時段，提供警力調派及民眾提早掌握防範


2. **犯罪類型分析**
   - 分析方向：追蹤特定犯罪類型（如毒品、竊盜、組織犯罪）的趨勢與風險
   - 衡量指標：各犯罪類型年犯罪率趨勢
   - 行動建議：
     - 毒品：強化查緝與反毒宣導
     - 竊盜：多留意住宅與財產安全
     - 搶奪/暴力：加強巡邏與治安部署
     - 組織犯罪：反詐騙宣導、持續搜集犯罪資料，以打擊幫派


4. **犯罪熱點分析**
   - 分析方向：找出長期高犯罪率縣市，以及治安改善最顯著的區域
   - 衡量指標：2024 v.s 2020 犯罪率百分比變化
   - 行動建議：建立縣市治安紅綠燈，作為居住遷移的風險參考
     - 熱點縣市犯罪率上升 → 提醒租屋/購屋族避開
     - 熱點縣市犯罪率下降 → 可考慮居住，但需實地調查


5. **雙北治安與房價交叉分析**
   - 分析雙北縣市治安風險與購/租屋成本的關聯，尋找「性價比」最高的行政區
   - 決策方向：
     - 犯罪率⬆︎、房價⬇︎ → 房價低但風險高
     - 犯罪率⬆︎、房價⬆︎ → 治安差又房價高，不建議
     - 犯罪率⬇︎、房價⬇︎ → 治安好又房價低，優先考慮
     - 犯罪率⬇︎、房價⬆︎ → 治安佳但價格高，適合預算充足者

## 資料處理流程
   - 整合各年度、季度犯罪資料為單一檔案
   - 年份轉換：民國 → 西元
   - 日期欄位合併：年 / 月 / 日 → date
   - 縣市名稱格式統一（台 → 臺）
