# Crime data analysis in Taiwan (2020–2024)

本專案針對台灣 2020–2025 年間的犯罪資料進行探索與分析，並結合人口與地區特徵，觀察犯罪趨勢及區域差異，最後整理成可視化圖表與簡報建議。

---

## 專案介紹
- **研究目的**：觀察近五年台灣犯罪率狀況，作為租屋、購屋族以及相關單位治安部署參考
- **資料來源**：犯罪資料、人口統計資料、新北臺北實價登錄
- **研究工具**：BigQuery、Google Sheets、Excel、Google Slides
- **使用情境**：
  - BigQuery：進行 SQL 資料查詢
  - Google Sheets、Excel：製作圖表
  - Google Slides：製作分析簡報
- **主要發現**：
  - 治安五年趨近穩定，但 113 年反彈，住宅竊盜、強制性交需提高警戒，需針對熱點加強防治、連假前超前部署
  - 基隆、桃園、嘉義市為近五年犯罪率前三高，近年再度升溫，犯罪以毒品與竊盜為主，需強化宣導與在地防治
  - 租屋購屋族需避開萬華、三重區，北投、文山、內湖與士林區性價比高，可以優先考慮

## 分析前架構大綱
1. **年度犯罪趨勢**
   - 我想知道什麼？
     - 每年犯罪率是否有上升或下降趨勢
   - 為何需要知道？
     - 上升 → 代表治安風險密度惡化
     - 下降 → 代表治安風險密度改善
   - 知道後能怎麼做？
     - 上升 → 針對熱點區域加強巡邏與監控措施
     - 下降 → 目前治安管理可能有效，可思考該年度是否有增加巡邏密度等，進一步評估成功原因，並將這些策略應用到其他犯罪率較高的區域
   - 有什麼衡量依據？
     - 年
     - 總案件
     - 總人口
     - 犯罪率（總案件數 ÷ 總人口 × 10,000，每萬人）


2. **犯罪月分佈**
   - 我想知道什麼？
     - 春節、寒暑假是否容易出現犯罪高峰？
   - 為何需要知道？
     - 判斷高風險時段，民眾可提早掌握或請相關單位提升防範意識
   - 知道後能怎麼做？
     - 提早強化預防、調派警力
     - 提早教育民眾（年前防竊、寒假反毒等）
   - 有什麼衡量依據？
     - 犯罪月份
     - 總案件


3. **犯罪類型趨勢**
   - 我想知道什麼？
     - 每年各類型犯罪率是否有上升或下降趨勢
   - 為何需要知道？
     - 毒品上升 → 毒品流通熱區擴大
     - 竊盜上升 → 人身財產安全風險提高
     - 搶奪/暴力上升 → 人身安全風險提高
     - 組織犯罪上升 → 幫派或犯罪集團活動活躍
   - 知道後能怎麼做？
     - 毒品 → 強化查緝與教育宣導
     - 竊盜 → 留意住宅與財產防護
     - 搶奪/暴力 → 加強日夜巡邏與治安部署
     - 組織犯罪 → 與警政單位合作，加強情資蒐集與打擊幫派
   - 有什麼衡量依據？
     - 年
     - 犯罪類型
     - 各年度總案件
     - 各年度總人口
     - 年度犯罪率（總案件數 ÷ 總人口 × 10,000，每萬人）


4. **各縣市犯罪率分布 ＆ 犯罪熱點分析**
   - 我想知道什麼？
     - 哪個縣市犯罪率下降幅度高，找出治安改善高的縣市
     - 哪個縣市犯罪率長期偏高，找出犯罪熱點
     - 犯罪熱點縣市是否有上升或下降趨勢
   - 為何需要知道？
     - 犯罪率下降幅度高 → 治安有持續改善
     - 犯罪率長期偏高 → 不適合租屋/購屋族
   - 知道後能怎麼做？
     - 改善縣市 → 分析成功原因並複製到高風險地區
     - 熱點縣市上升 → 提醒租屋/購屋族避開
     - 熱點縣市下降 → 可考慮居住，但需實地調查
   - 有什麼衡量依據？
     - 年
     - 縣市
     - 五年總人口
     - 五年總案件數
     - 犯罪率（總案件數 ÷ 總人口 × 10,000，每萬人）
     - 2024 v.s 2020 犯罪率百分比變化（2024 犯罪率 / 2020 犯罪率 - 1）


5. **雙北犯罪率 v.s 房價交叉分析**
   - 我想知道什麼？
     - 治安與居住成本的關係
   - 為何需要知道？
     - 幫助租屋或購屋族綜合考量房價與治安風險，做出更全面的選擇
   - 知道後能怎麼做？
     - 犯罪率⬆︎、房價⬇︎ → 房價低但風險高
     - 犯罪率⬆︎、房價⬆︎ → 治安差又房價高，不建議
     - 犯罪率⬇︎、房價⬇︎ → 治安好又房價低，優先考慮
     - 犯罪率⬇︎、房價⬆︎ → 治安佳但價格高，適合預算充足者
   - 有什麼衡量依據？
     - 雙北行政區
     - 五年每坪單價（中位數）
     - 五年總人口
     - 五年犯罪總數
     - 犯罪率（總案件數 ÷ 總人口 × 10,000，每萬人）

   - 民國 → 西元轉換
   - 整併年、月、日欄位一欄 date 
   - 縣市名稱格式統一（台 → 臺）
